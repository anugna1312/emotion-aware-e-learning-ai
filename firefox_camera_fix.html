🦊 Firefox Camera Emotion Capture

🔥 Firefox-Specific Instructions:

1. Click Start Camera button below

2. When Firefox asks for permission, click "Allow"

3. If permission denied, click 🔒 icon in address bar → Camera → "Allow"

4. Refresh page and try again



🛠️ Troubleshooting:

❌ If camera doesn't start:

  • Click 🔒 icon in address bar
  • Set Camera to "Allow"
  • Refresh page (F5)
  • Try again

❌ If still not working:

  • Close all Firefox tabs except this one
  • Restart Firefox completely
  • Try different browser (Chrome/Edge)
